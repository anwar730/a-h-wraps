# A&H Manufacturing Website

Vue 3 + Tailwind CSS website for A&H Manufacturing, LLC — US-made stretch wrap film, Atlanta, GA.

## Pages

| Route | Description |
|-------|-------------|
| `/` | Homepage — hero, value props, products snapshot, who we serve, testimonials |
| `/about` | Company story, founder bio, why domestic manufacturing |
| `/products` | Product details + comparison table |
| `/contact` | Contact form + business info |

## Tech Stack

- **Vue 3** (Composition API + `<script setup>`)
- **Vue Router 4** (history mode)
- **Tailwind CSS 3**
- **Vite 5**

## Setup

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
